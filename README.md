# Lorefold — Public Release Assets

Public mirror of release artifacts for [Lorefold](https://gestaltic.org) (the
product formerly named Engram). Source code lives in a private repository; this
repo exists only to host the Windows installer so the desktop app's update
check has a reachable download URL.

The installer keeps its long-standing filename (`Engram-Setup.exe`) so existing
installs and update flows keep working.

New here? Start at [gestaltic.org/install](https://gestaltic.org/install) rather
than downloading directly from this page.

Download the latest installer from the [Releases](../../releases) page.
